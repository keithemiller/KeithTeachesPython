Welcome to Keith Teaches Python

This is an introductory overview of Python, but what is Python? Python refers to "CPython" which is a version of the Python interpreter built in C. Some other implementations include Jython which is built in Java and IronPython which is built in C#. Also, there is in fact a difference between certain versions of Python. Python3 and Python2 have some key differences in how they were built. Today, when I say Python, I mean Python2 not Python3. Python3 is much newer, but Python2 is still dominant.
